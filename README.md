# ssg

A personal project to build a modern static site generator from first principles, with a strong path for long-term extensibility.

This repo starts with a practical plan and will evolve toward a production-capable SSG that can support:
- Markdown/MDX content workflows
- Templating and layouts
- Asset pipelines
- Plugin/extensibility hooks
- Modern frontend integrations (WebGL/WebGPU/WASM)

## Status
- Bootstrap phase: planning and architecture baseline
- CI is now set up for repository validation (plan/repo basics)
- Next: implement a minimal TypeScript-powered MVP generator
